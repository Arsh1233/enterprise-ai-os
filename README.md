# Enterprise AI Operating System (EAIOS)

## Project Overview
The Enterprise AI Operating System (EAIOS) is a comprehensive, production-grade monorepo designed to build and operate robust AI-powered enterprise applications.

## Architecture Overview
The system is divided into scalable backend services and frontend applications. The architecture employs microservices principles orchestrated via Kubernetes, and utilizes AWS for cloud infrastructure.

## Folder Structure
- `backend/` - Python/FastAPI based backend services.
- `frontend/` - Next.js based frontend applications.
- `docs/` - Architecture, API, and development documentation.
- `design/` - Design assets and specifications.
- `infrastructure/` - Docker, Kubernetes, Terraform, and Helm configurations.
- `scripts/` - Utility scripts for setup, development, and deployment.
- `.github/` - GitHub Actions workflows and configuration.

## Technology Stack
- **Backend:** Python 3.12, FastAPI, Poetry
- **Frontend:** Next.js
- **Infrastructure:** Docker, Docker Compose, Kubernetes, Terraform, AWS
- **AI/ML:** Google ADK, LangGraph, MCP
- **Databases:** PostgreSQL, Redis, Neo4j, Qdrant

## Prerequisites
- Python 3.12+
- Node.js & npm/yarn
- Docker & Docker Compose
- Terraform
- Poetry (for Python dependency management)
